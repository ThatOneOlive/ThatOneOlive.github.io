Below are some projects of mine that I think are interesting, or otherwise pass my personal test of quality.

Rowan
Rowan is a small assembly language and virtual machine runtime. Essentially, the virtual machine executes a custom binary format, and Rowan the assembly language is translated into this format. Rowan supports common assembly needs such as a stack and subroutines.
Language: C++
Lithos
Lithos is a work-in-progress 32-bit kernel for x86, developed with the intent to become more familiar with low-level operating system workings more rigorously.
Languages: C, Assembly(x86)
